my road to web development

Hello.

I study full-stack web development, creating web applications from the ground up. I am currently working on projects to hone my skills and be a better web developer and designer.

Developer

I enjoy web development from the ground up and learning new techniques and technologies. I started by learning python as a hobby and found that I had a passion for coding. After finishing a Python course, I began learning HTML, CSS, and JavaScript. I picked up express and paired it with MongoDB to create my first web application. I will continue to learn and grow as a developer each day.

Python

Python is the first programming language that I studied. I really like the versitility and readablity of this language

HTML:5

I use all the latest and accepted syntax with HTML 5 to ensure screen readers and search results are optimized

CSS

I enjoy styling with CSS, and I am always learning new tricks and techniques as well as keeping up with the latest CSS updates and methods

JavaScript

JavaScript is the language the popular browsers intereract with for any web application. I keep up with all updates to ensure best practices are being used

NodeJS

I learned NodeJS paired with an express server. The ability to run asynchronous functions makes it a powerful choice

Adobe Photoshop

Adobe Photoshop is an essential tool that I use for designing web applications, graphics, and images

PostgreSQL

PostgreSQL is the first database I learned. Knowing how to set up a database securely is a vital part of web development

MongoDB

MongoDB is a no SQL database that uses JavaScript as its query language. The versatility of no SQL databases is limitless
